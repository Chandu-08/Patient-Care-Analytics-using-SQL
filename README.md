# 🏥 Healthcare Analytics using SQL Server

## 📖 Overview

This project analyzes hospital admission data using Microsoft SQL Server to generate meaningful healthcare insights. The workflow includes data cleaning, duplicate removal, KPI calculation, and patient demographic analysis.

## 🎯 Objectives

- Clean and prepare healthcare data
- Remove duplicate patient records
- Calculate healthcare KPIs
- Analyze discharge trends
- Evaluate patient demographics and outcomes

## 🛠️ Tools & Technologies

- Microsoft SQL Server
- SQL Server Management Studio (SSMS)
- T-SQL
- Views
- CTEs
- Window Functions

## 📊 Key Insights

- Total Discharges
- Average Length of Stay (ALOS)
- Discharge Rate Analysis
- Age Group Distribution
- Gender Distribution
- Day-wise Discharge Trends

## 📂 Files

- **Data Prep.sql** – Data cleaning and preparation
- **Health Care Answer.sql** – KPI calculations and business analysis


## 📌 Project Outcome

Transformed raw hospital admission data into actionable insights that support healthcare performance monitoring and data-driven decision-making.
